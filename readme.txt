E-commerce delivery with real-time tracking.

== Description ==

WooCommerce shipping method with real-time delivery tracking, external application and administrative order creation
intended exclusively for the Brazilian public.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install plugin through the WordPress plugins screen directly.
2. Activate the plugin through the plugins screen in WordPress.
3. Use the WooCommerce -> Settings -> Shipping -> Uptogo screen to configure the plugin.

== Frequently Asked Questions ==

= What delivery regions are served? =

Only the Brazilian territory.

= How do I get an access key? =

You can view your access key on the settings screen of our app.

= How do I change the store address? =

You can change the store address in the settings screen of our app.
